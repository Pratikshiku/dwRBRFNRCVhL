### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`/`MybatisPlus`/`jdk1.8`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`/`ElementUI`/`富文本`等。
- 数据库：`MySQL`/`MybatisPlus`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：学生、管理员、教师。
主要功能：前台（用学生，管理员都能使用），后台管理（仅供管理员、教师才能使用）

**前台（用学生，管理员都能使用）功能：**
1. 登录
2. 注册
3. 安全退出
4. 查看作业
5. 社区问答
6. 练习
7. 在线学习
8. 班级管理
9. 申请班级记录
10. 习题提交记录
11. 基本信息
12. 补充得知识点

**后台管理（仅供管理员、教师才能使用）功能：**
1. 登录
2. 学生管理
3. 教师管理
4. 作业管理
5. 科目管理
6. 班级管理
7. 试题管理
8. 知识点管理
9. 个人信息管理
